# 스페셜포스 리마스터 공식 웹사이트

PDF 기획안과 디자인 목업을 기반으로 한 웹페이지 구축 프로젝트입니다.

## 🎮 프로젝트 소개

스페셜포스 리마스터 게임의 공식 웹사이트로, 게임 정보, 새소식, 미디어, 고객센터 등 다양한 콘텐츠를 제공합니다.

## 📁 프로젝트 구조

```
SF-Remaster-proj/
├── assets/              # 이미지, 폰트 등 정적 리소스
│   ├── images/
│   ├── fonts/
│   └── videos/
├── docs/                # 프로젝트 문서
│   ├── PDF분석결과.md
│   ├── 구축계획서.md
│   └── 구현가이드.md
├── src/                 # 소스 코드
│   ├── css/
│   │   ├── variables.css  # 디자인 시스템 변수
│   │   ├── reset.css      # CSS 리셋
│   │   └── style.css      # 메인 스타일
│   ├── js/
│   │   └── main.js        # 메인 스크립트
│   └── html/
│       └── index.html     # 메인 페이지
├── dist/                # 빌드 결과물
├── vercel.json          # Vercel 배포 설정
└── README.md
```

## 🚀 시작하기

### 로컬 개발

1. 이 저장소를 클론하세요:
```bash
git clone [your-repo-url]
cd SF-Remaster-proj
```

2. 브라우저에서 `src/html/index.html`을 열어 확인하세요.

### 배포

이 프로젝트는 [Vercel](https://vercel.com)에서 배포됩니다.

## 🛠 기술 스택

- **HTML5** - 시맨틱 마크업
- **CSS3** - 반응형 디자인, CSS Variables
- **JavaScript (ES6+)** - 인터랙티브 기능
- **Pretendard** - 웹폰트

## 🎨 디자인 시스템

- **Primary Color**: #46E57E (RGB: 0.274, 0.899, 0.494)
- **폰트**: Pretendard
- **반응형 브레이크포인트**: 
  - Mobile: ~768px
  - Tablet: 769px ~ 1024px
  - Desktop: 1025px+

## 📄 주요 페이지

- **메인 페이지** (`/`) - 히어로 섹션, 새소식, 게임 특징
- **새소식** (`/news`) - 게임 공지사항, 업데이트, 이벤트
- **게임정보** (`/game-info`) - IP, 맵, 무기, 캐릭터, 가이드
- **미디어** (`/media`) - 영상, 아카이브
- **고객센터** (`/support`) - FAQ, 1:1 문의

## 🖼 이미지 리소스

이미지를 추가하려면 `assets/images/` 폴더 구조를 참고하세요:
- `hero/` - 히어로 섹션 이미지
- `logo/` - 로고 파일
- `features/` - 게임 특징 이미지 (4개)
- `news/` - 새소식 썸네일
- `icons/` - 플랫폼 아이콘

각 폴더에 README 파일이 있어 상세 가이드를 제공합니다.

## 🔧 주요 기능

- **반응형 디자인**: 모바일, 태블릿, 데스크톱 완벽 지원
- **인터랙티브 UI**: 스무스 스크롤, 헤더 스크롤 효과, 카드 호버 애니메이션
- **모바일 메뉴**: 햄버거 메뉴로 모바일 환경 최적화
- **모달 팝업**: 게임 시작 플랫폼 선택 모달
- **이미지 처리**: 자동 placeholder 처리 및 오류 핸들링
- **접근성**: 시맨틱 HTML, ARIA 속성, 키보드 네비게이션 지원
