# 새소식 썸네일 이미지

이 폴더에는 새소식 섹션에서 사용하는 썸네일 이미지들이 배치됩니다.

## 필요한 이미지

### PDF에서 추출할 이미지:
- 새소식 카드별 썸네일 이미지 (4개 이상)

## 사용 위치

- `src/html/index.html` - News Section
  ```html
  <img src="../../assets/images/news/[filename].jpg" alt="새소식 썸네일">
  ```

## 이미지 명명 규칙

- `news-001.jpg`, `news-002.jpg` 등으로 순서대로 저장
- 또는 의미있는 이름: `update-20260326.jpg` 등

## 이미지 추출 방법

1. PDF에서 새소식 섹션 페이지 확인
2. 각 새소식 카드의 썸네일 이미지 추출
3. JPG 형식으로 저장
4. 권장 해상도: 최소 400x300px (4:3 비율)
