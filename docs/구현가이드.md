# SF 리마스터 웹사이트 구현 가이드

## 🚀 빠른 시작

### 1. 폰트 설정

PRETENDARD 폰트를 설정합니다.

**방법 1: CDN 사용 (권장)**
```html
<link rel="stylesheet" as="style" crossorigin href="https://cdn.jsdelivr.net/gh/orioncactus/pretendard@v1.3.9/dist/web/static/pretendard.min.css" />
```

**방법 2: 로컬 파일 사용**
1. [Pretendard 다운로드](https://github.com/orioncactus/pretendard/releases)
2. `assets/fonts/pretendard/` 폴더에 저장
3. CSS에서 @font-face 정의

### 2. 기본 CSS 변수 설정

```css
:root {
  /* 컬러 */
  --color-primary: #46E57E; /* RGB(0.274, 0.899, 0.494) */
  --color-text-primary: #000000;
  --color-text-secondary: #666666;
  --color-bg: #ffffff;
  
  /* 폰트 */
  --font-family: 'Pretendard', -apple-system, BlinkMacSystemFont, system-ui, sans-serif;
  
  /* 간격 */
  --spacing-xs: 4px;
  --spacing-sm: 8px;
  --spacing-md: 16px;
  --spacing-lg: 24px;
  --spacing-xl: 32px;
  --spacing-2xl: 48px;
  
  /* 브레이크포인트 */
  --breakpoint-mobile: 768px;
  --breakpoint-tablet: 1024px;
  --breakpoint-desktop: 1025px;
}
```

## 🎨 주요 구현 포인트

### 1. 영상 백그라운드 (메인 페이지)

```html
<div class="hero-section">
  <video autoplay muted loop playsinline>
    <source src="assets/videos/hero-bg.mp4" type="video/mp4">
  </video>
  <div class="hero-content">
    <!-- 콘텐츠 -->
  </div>
</div>
```

```css
.hero-section {
  position: relative;
  height: 100vh;
  overflow: hidden;
}

.hero-section video {
  position: absolute;
  top: 50%;
  left: 50%;
  transform: translate(-50%, -50%);
  min-width: 100%;
  min-height: 100%;
  width: auto;
  height: auto;
  z-index: 1;
}

.hero-content {
  position: relative;
  z-index: 2;
}
```

### 2. 네비게이션 롤오버

```css
.nav-item {
  position: relative;
}

.nav-item:hover .submenu {
  display: block;
  opacity: 1;
  transform: translateY(0);
}

.submenu {
  display: none;
  position: absolute;
  top: 100%;
  left: 0;
  opacity: 0;
  transform: translateY(-10px);
  transition: all 0.3s ease;
}
```

### 3. 게임 특징 카드 호버

```css
.feature-card {
  transition: transform 0.3s ease;
}

.feature-card:hover {
  transform: translateY(-10px);
}

.feature-card:hover .feature-detail {
  display: block;
}
```

### 4. 필터 기능

```javascript
function filterNews(category) {
  const items = document.querySelectorAll('.news-item');
  items.forEach(item => {
    if (category === 'all' || item.dataset.category === category) {
      item.style.display = 'block';
    } else {
      item.style.display = 'none';
    }
  });
}
```
