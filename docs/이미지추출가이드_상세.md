# PDF 이미지 추출 상세 가이드

이 문서는 PDF에서 필요한 이미지들을 추출하고 프로젝트에 배치하는 방법을 안내합니다.

## 📁 폴더 구조

```
assets/
├── images/
│   ├── hero/              # 히어로 섹션 이미지
│   │   └── main-hero.png
│   ├── logo/              # 로고 파일
│   │   ├── shield-logo.png
│   │   └── dragonfly-logo.png
│   ├── features/          # 게임 특징 이미지
│   │   ├── level-design.jpg
│   │   ├── moving-control.jpg
│   │   ├── shooting-system.jpg
│   │   └── weapon-system.jpg
│   ├── news/              # 새소식 썸네일
│   │   └── news-*.jpg
│   └── icons/             # 아이콘
│       ├── steam-icon.png
│       └── epic-icon.png
└── videos/
    └── hero-bg.mp4        # 히어로 배경 영상
```

## 🖼 이미지별 추출 가이드

### 1. 히어로 섹션 이미지

**위치**: PDF 메인 페이지 (첫 페이지)
**파일명**: `main-hero.png`
**대상 폴더**: `assets/images/hero/`

**추출 방법**:
1. PDF 메인 페이지의 히어로 섹션 확인
2. 대표 이미지 부분 선택 및 추출
3. PNG 형식으로 저장
4. 해상도: 최소 1920x1080px 권장

## 🛠 이미지 추출 도구

### 방법 1: Python 스크립트 (자동화)
```bash
# 필요한 라이브러리 설치
pip install PyMuPDF

# 스크립트 실행
python scripts/extract_pdf_images.py
```

### 방법 2: Adobe Acrobat
1. Adobe Acrobat에서 PDF 열기
2. 도구 > 페이지 구성 선택
3. 이미지 선택 > 우클릭 > 이미지로 저장

### 방법 3: 온라인 도구
- [SmallPDF](https://smallpdf.com/pdf-to-jpg)
- [ILovePDF](https://www.ilovepdf.com/pdf-to-jpg)
- [PDF24](https://tools.pdf24.org/ko/pdf-to-jpg)

**주의**: 민감한 정보가 포함된 경우 온라인 도구 사용 시 주의하세요.

### 방법 4: 스크린샷
1. PDF 뷰어에서 필요한 페이지 열기
2. Windows: Win + Shift + S (스니핑 도구)
3. Mac: Cmd + Shift + 4
4. 이미지 편집 도구에서 필요한 부분만 자르기
5. 적절한 폴더에 저장

## ✅ 추출 후 확인사항

### 1. 파일명 확인
- 모든 이미지 파일명이 HTML에 지정된 경로와 일치하는지 확인
- 대소문자 구분 확인 (Windows는 대소문자 구분 안 함)

### 2. 파일 형식 확인
- 로고/아이콘: PNG (투명 배경)
- 일반 이미지: JPG
- 영상: MP4

### 3. 해상도 확인
- 웹용 최적화 확인 (불필요하게 큰 파일 크기 방지)
- 이미지 압축 도구 사용 권장

### 4. HTML 경로 확인
`src/html/index.html`에서 이미지 경로가 올바른지 확인:
- 상대 경로: `../../assets/images/...`
- 절대 경로가 아닌 상대 경로 사용

## 📞 문제 해결

### 이미지를 찾을 수 없는 경우:
1. HTML에서 이미지 경로 확인
2. 파일명이 정확한지 확인
3. 파일이 올바른 폴더에 있는지 확인

### 이미지가 표시되지 않는 경우:
1. 브라우저 개발자 도구에서 에러 확인
2. 파일 경로 확인 (대소문자, 확장자)
3. 서버에서 해당 경로 접근 가능한지 확인
