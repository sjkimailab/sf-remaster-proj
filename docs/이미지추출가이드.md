# PDF 이미지 추출 가이드

## 🖼 이미지 추출 방법

PDF에서 이미지를 추출하는 여러 방법을 안내합니다.

### 방법 1: Python 스크립트 사용 (권장)

#### 필요한 라이브러리 설치
```bash
pip install PyMuPDF pdf2image
```

Windows의 경우 pdf2image 사용 시 poppler도 필요합니다:
- [Poppler for Windows 다운로드](https://github.com/oschwartz10612/poppler-windows/releases/)
- 다운로드 후 PATH에 추가

#### 스크립트 실행
```bash
python scripts/extract_pdf_images.py
```

추출된 이미지는 `assets/images/other/` 폴더에 저장됩니다.

---

### 방법 2: Adobe Acrobat 사용

1. Adobe Acrobat에서 PDF 열기
2. **도구 > 페이지 구성** 선택
3. **페이지 구성** 패널에서 이미지 선택
4. 이미지 우클릭 > **이미지로 저장**

---

### 방법 3: 온라인 도구 사용

다음 온라인 도구를 사용할 수 있습니다:
- [SmallPDF](https://smallpdf.com/pdf-to-jpg)
- [ILovePDF](https://www.ilovepdf.com/pdf-to-jpg)
- [PDF24](https://tools.pdf24.org/ko/pdf-to-jpg)

**주의**: 민감한 정보가 포함된 경우 온라인 도구 사용 시 주의하세요.

---

### 방법 4: 스크린샷 사용

1. PDF 뷰어에서 페이지 열기
2. 필요한 이미지/디자인 부분 스크린샷
3. 이미지 편집 도구에서 필요한 부분만 자르기
4. `assets/images/` 적절한 폴더에 저장

---

### 방법 5: Windows의 경우 - Snipping Tool 사용

1. PDF 파일 열기
2. Windows 키 + Shift + S로 스니핑 도구 실행
3. 원하는 영역 선택
4. 이미지 저장
5. `assets/images/` 적절한 폴더에 저장

---

## 📁 이미지 정리

추출한 이미지를 다음 구조로 정리하세요:

```
assets/images/
├── hero/              # 히어로 섹션 이미지
│   └── hero-bg.jpg
├── logo/              # 로고 파일
│   └── logo.png
├── backgrounds/       # 배경 이미지
│   └── bg-pattern.jpg
├── icons/             # 아이콘
│   └── [아이콘 파일들]
└── other/             # 기타 이미지
    └── [추출된 이미지들]
```

## 🎨 이미지 최적화 (선택사항)

추출한 이미지를 웹 최적화하기:

### 온라인 도구
- [TinyPNG](https://tinypng.com/) - PNG/JPG 압축
- [Squoosh](https://squoosh.app/) - 이미지 최적화

### 명령줄 도구
```bash
# ImageMagick 사용 (설치 필요)
convert input.jpg -quality 85 -resize 1920x output.jpg

# 또는 optipng 사용
optipng -o7 *.png
```
