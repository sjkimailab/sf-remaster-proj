# SF 리마스터 웹사이트 페이지 구조도

## 사이트 맵

```
/ (메인 페이지)
├── /news (새소식)
│   ├── /news/list (리스트)
│   └── /news/detail/[id] (상세)
│
├── /game-info (게임정보)
│   ├── /game-info/ip (스페셜포스 IP)
│   ├── /game-info/maps (맵)
│   │   └── /game-info/maps/[map-name] (맵 상세)
│   ├── /game-info/weapons (무기)
│   │   └── /game-info/weapons/[weapon-name] (무기 상세)
│   ├── /game-info/characters (캐릭터)
│   │   └── /game-info/characters/[character-name] (캐릭터 상세)
│   └── /game-info/guides (기본가이드)
│       └── /game-info/guides/[guide-id] (가이드 상세)
│
├── /media (미디어)
│   ├── /media/videos (영상)
│   │   └── /media/videos/[id] (영상 재생)
│   └── /media/archive (아카이브)
│       └── /media/archive/[id] (아카이브 이미지)
│
├── /support (고객센터)
│   ├── /support/faq (FAQ)
│   │   └── /support/faq/[id] (FAQ 상세)
│   └── /support/inquiry (1:1 문의)
│
└── /terms (이용약관)
    ├── /terms/privacy (개인정보 처리방침)
    ├── /terms/service (서비스 이용약관)
    └── /terms/policy (운영정책)
```

## 네비게이션 구조

### 메인 메뉴
1. **새소식** (/news)
   - 서브메뉴 없음

2. **게임정보** (/game-info)
   - 서브메뉴:
     - 스페셜포스 IP
     - 맵
     - 무기
     - 캐릭터
     - 기본가이드

3. **미디어** (/media)
   - 서브메뉴:
     - 영상
     - 아카이브

4. **고객센터** (/support)
   - 서브메뉴:
     - FAQ
     - 1:1 문의

### 특수 요소
- **GAME START** 버튼: 게임 시작 팝업 표시

## 주요 컴포넌트

### 공통 컴포넌트
1. **Header** (모든 페이지)
   - 로고
   - 네비게이션
   - GAME START 버튼

2. **Footer** (모든 페이지)
   - 회사 정보
   - 약관 링크
   - 소셜 링크
   - 저작권

### 페이지별 컴포넌트
1. **메인 페이지**
   - 히어로 섹션
   - 새소식 섹션
   - 게임 특징 섹션

2. **리스트 페이지**
   - 필터/탭
   - 카드 리스트
   - 페이지네이션

3. **상세 페이지**
   - 제목
   - 날짜/분류
   - 내용
   - 목록 버튼

4. **팝업/모달**
   - 게임 시작 팝업
   - 이미지 뷰어
   - 영상 플레이어
