```
│   │   ├── index.html         # 메인 페이지

│   ├── css/                   # CSS 파일들

│   └── js/                    # JavaScript 파일들
│       ├── main.js            # 메인 JavaScript
│       └── [modules].js       # 모듈별 JavaScript
└── dist/                      # 빌드 결과물
```

## 🛠 기술 스택

### Frontend
- **HTML5**: 시맨틱 마크업
- **CSS3**: 
  - CSS Grid & Flexbox (레이아웃)
  - CSS Variables (컬러 시스템)
  - CSS Animations (인터랙션)
- **JavaScript (ES6+)**:
  - 바닐라 JavaScript (프레임워크 없이)
  - 모듈화 구조

### 개발 도구 (선택사항)
- **빌드 도구**: 필요시 Vite 또는 Webpack
- **CSS 전처리기**: 필요시 Sass/SCSS
- **버전 관리**: Git

## 📐 디자인 시스템 (PDF에서 추출 예정)

### 컬러 팔레트
- Primary Color: RGB(0.274, 0.899, 0.494) - HEX: #46E57E (게임 특징 컬러)
- Background Colors: 영상 배경 (메인 페이지)
- Text Colors: 기본 텍스트 색상

### 타이포그래피
- **폰트**: PRETENDARD
  - 웹폰트 사용 (CDN 또는 로컬 파일)
  - [Pretendard 다운로드](https://codedragon.org/)
- Font Sizes: (디자인 목업 기준으로 추후 확정)

### 간격 시스템
- Spacing Scale: 4px / 8px / 16px / 24px / 32px / 48px / 64px

### 브레이크포인트
- Mobile: ~768px
- Tablet: 769px ~ 1024px
- Desktop: 1025px+

## 🎨 페이지 구조 (기획서 분석 후 확정)

### 페이지 목록
1. **메인 페이지** (index.html)
   - 히어로 섹션 (영상 배경, 무한 재생)
   - 새소식 섹션 (최신 4개)
   - 게임 특징 섹션 (4개 카드)
   - 푸터

2. **새소식 페이지** (news.html)
   - 리스트 페이지 (필터: 전체/공지/업데이트/이벤트)
   - 상세 페이지

3. **게임정보 페이지** (game-info.html)
   - 스페셜포스 IP
   - 맵 (리스트/상세)
   - 무기 (리스트/상세)
   - 캐릭터 (리스트/상세)
   - 기본가이드 (리스트/상세)

4. **미디어 페이지** (media.html)
   - 영상 (리스트/재생)
   - 아카이브 (리스트/이미지 뷰어)

5. **고객센터 페이지** (support.html)
   - FAQ (리스트/상세)
   - 1:1 문의하기 (폼)

6. **이용약관 페이지** (terms.html)
   - 개인정보 처리방침
   - 서비스 이용약관
   - 운영정책
